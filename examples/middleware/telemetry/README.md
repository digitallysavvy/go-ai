# Telemetry Middleware

Collect metrics and telemetry data from AI operations.

## Features

- Request tracking
- Performance metrics
- Token usage monitoring
- Success/error rates

## Quick Start

```bash
export OPENAI_API_KEY=sk-...
go run main.go
```

## Metrics

- request_count
- request_duration_ms
- tokens_used
- success_count
- error_count
