# Multi-Agent System

Multiple AI agents working together on complex tasks.

## Features

- Agent specialization
- Sequential workflow
- Role-based agents

## Agents

1. **Researcher** - Gathers information
2. **Writer** - Creates content
3. **Reviewer** - Improves quality

## Quick Start

```bash
export OPENAI_API_KEY=sk-...
go run main.go
```
