# xAI Provider

xAI provides Grok models with real-time knowledge through X (Twitter) integration. Known for witty personality, current events awareness, and strong reasoning capabilities.

## Setup

### Installation

```go
import (
    "github.com/digitallysavvy/go-ai/pkg/ai"
    "github.com/digitallysavvy/go-ai/pkg/providers/openai" // xAI uses OpenAI-compatible API
)
```

### Configuration

```go
provider := openai.New(openai.Config{
    APIKey:  os.Getenv("XAI_API_KEY"),
    BaseURL: "https://api.x.ai/v1",
})

model, err := provider.LanguageModel("grok-2-latest")
if err != nil {
    log.Fatal(err)
}
```

### Get API Key

1. Sign up at [x.ai](https://x.ai)
2. Request API access
3. Set environment variable:

```bash
export XAI_API_KEY=xai-...
```

## Provider-Specific Features

### Real-Time X Integration

Access current events and trending topics:

```go
result, err := ai.GenerateText(model,
    "What are the trending topics on X today?",
    ai.WithWebSearch(true), // Enable real-time X data
)
```

### Vision Capabilities

```go
result, err := ai.GenerateText(model,
    ai.WithMessages(
        ai.UserMessage("Describe this meme",
            ai.WithImageURL("https://example.com/meme.jpg"),
        ),
    ),
)
```

### Tool Use

```go
searchTool := ai.Tool{
    Type: "function",
    Function: ai.FunctionDefinition{
        Name: "search_x",
        Description: "Search X posts",
        Parameters: map[string]interface{}{
            "type": "object",
            "properties": map[string]interface{}{
                "query": map[string]string{"type": "string"},
            },
        },
    },
}

result, err := ai.GenerateText(model,
    "Find recent AI discussions",
    ai.WithTools(searchTool),
)
```

## Examples

### Basic Text Generation

```go
package main

import (
    "fmt"
    "log"
    "os"

    "github.com/digitallysavvy/go-ai/pkg/ai"
    "github.com/digitallysavvy/go-ai/pkg/providers/openai"
)

func main() {
    provider := openai.New(openai.Config{
        APIKey:  os.Getenv("XAI_API_KEY"),
        BaseURL: "https://api.x.ai/v1",
    })

    model, err := provider.LanguageModel("grok-2-latest")
    if err != nil {
        log.Fatal(err)
    }

    result, err := ai.GenerateText(model,
        "What's happening in AI today?")
    if err != nil {
        log.Fatal(err)
    }

    fmt.Println(result.Text)
}
```

### Current Events Analysis

```go
result, err := ai.GenerateText(model,
    "Analyze today's tech news and give me key insights",
    ai.WithWebSearch(true),
)
```
